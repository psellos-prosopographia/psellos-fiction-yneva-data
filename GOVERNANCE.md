# Governance

Canonical governance policies live at repository root per the psellos-hub authority index.
This file defines local governance expectations for this dataset.

## Decision records

* Modeling decisions must be recorded in `DECISIONS/README.md` or in individual decision files referenced from it.
* Decisions are required for changes to assertion semantics, vocabularies, or rule regimes.

## Authority

The psellos-hub authority index is the external governance reference:
